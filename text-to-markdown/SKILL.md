# Text Formatting Guide

## Guide
Read the text file and format it to Markdown format.

**DO**
 - Use headings to make text structure more clear.
 - Use order or unordere lists as needed.
 - Use table.
 - Wrap inline equation with "$" and equation block with "$$". Check if equation follows Latex format or not. Change it to Latex format if needed.
 - Remove unnecessary blank space

**DO NOT**
 - Rewrite content. Just reformat it.
 - Use script to format it. Do it directly.

## Output
<original_file_name>-formatted.md
